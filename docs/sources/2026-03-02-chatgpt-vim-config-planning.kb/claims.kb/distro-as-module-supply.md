Distro modular ideas (lazy.nvim plugin groups, LazyVim Extras,
AstroCommunity packs) are valuable and reusable as module supply — copy or
import selected specs, rewrap into your own module contract. The distro
ecosystem becomes a supply chain, not a constitution.
